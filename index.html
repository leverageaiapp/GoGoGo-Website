Features

⚡

One-Command Setup

Run gogogo start and you're live. No configuration files, no accounts, no port forwarding.

📱

Mobile-First UI

Touch scrolling with inertia, dual input modes (textarea + native keyboard), and responsive layout.

🔒

Secure by Default

128-bit cryptographic token auth. httpOnly cookies. WebSocket verification. CSP and HSTS headers.

🌐

Cloudflare Tunnel

Uses Cloudflare Quick Tunnel for instant, secure HTTPS access. No account or configuration needed.

🔄

Reconnection Support

Output buffer with sequence tracking. Reconnecting clients get a delta sync — no lost output.

📸

Image Paste

Paste images from your clipboard. They're uploaded and the file path is inserted into your terminal.

How It Works

1

Install & Run

Install globally via npm, then run gogogo start (or gogogo start claude for a specific command).

2

Tunnel Created

GoGoGo starts a local server with a PTY and creates a Cloudflare Quick Tunnel for secure HTTPS access.

3

Scan QR Code

A QR code with your authenticated URL appears in the terminal. Scan it with your phone's camera.

4

Code from Your Pocket

Your terminal is now live on your phone. Type commands, scroll through output, paste images — all in real-time.
